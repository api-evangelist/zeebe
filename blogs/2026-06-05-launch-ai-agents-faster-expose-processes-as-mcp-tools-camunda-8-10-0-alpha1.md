Camunda 8.10.0-alpha1 is now live and available for download, with SaaS customers who stay current already seeing some features automatically deployed. Key additions in this release include the Processes MCP Server for exposing BPMN processes as MCP tools, a Camunda-provided LLM for faster agent setup, native ABBYY support, and an enhanced variable viewer in Operate. The alpha release is designed to reduce setup friction and accelerate AI agent development on the Camunda platform.
