Separating an agent's responsibilities into a decision-maker and an executor requires careful design-time awareness to keep these roles distinct, starting with understanding what the LLM component should and should not be asked to do. The post covers best practices for LLM selection and system prompt development when building production-grade Camunda agents. It is the second in a series on designing agents for production environments.
