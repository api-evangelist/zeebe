Getting started with AI Agents in Camunda has never been easier thanks to Camunda Provided LLM, which removes the common barriers of budget approval, IT security approval, and LLM credential setup. The Camunda-provided LLM is available immediately in the platform without any external configuration, allowing developers to run their first AI agent in minutes. The post walks through the quickstart experience for new and existing Camunda users exploring agentic orchestration.
